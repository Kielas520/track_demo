# Track Demo

基于 OpenCV 的实时目标跟踪演示系统，集成 HSV 颜色检测、CSRT/KCF 跟踪器和 3D 卡尔曼滤波。

## 功能

- **HSV 颜色检测**：滑动条实时调参，支持跨 0 度的红色范围
- **CSRT / KCF 跟踪器**：一键切换，自动重连丢失目标
- **3D 卡尔曼滤波**：对 (x, y, 面积) 平滑去噪 + 前馈预测补偿延迟
- **实时调参**：Q/R 噪声参数、预测模式、预测时间全部可调

## 安装

```bash
uv sync
```

或

```bash
pip install opencv-contrib-python>=4.13.0.92
```

## 运行

```bash
python main.py
```

## 操作

| 按键 | 功能 |
|------|------|
| `c` | 启动 CSRT 跟踪器（精度高） |
| `k` | 启动 KCF 跟踪器（速度快） |
| `r` | 重置跟踪 |
| `q` | 退出 |

## 可视化说明

| 标记 | 颜色 | 含义 |
|------|------|------|
| 空心矩形 | 绿色 | 跟踪器原始 bbox |
| 实心圆 | 蓝色 | 卡尔曼滤波平滑位置 |
| 空心圆 | 红色 | 前馈预测位置 |
| 高亮区域 | 黄色 | ROI 内 HSV 掩膜匹配区 |
| 空心矩形 | 黄色 | 待选目标（未跟踪时） |

## 滑动条

### Tracker Main 窗口

| 滑动条 | 范围 | 说明 |
|--------|------|------|
| Q_x, Q_y, Q_z | 0.01 - 10.0 | 位置/面积过程噪声 |
| Q_vx, Q_vy, Q_vz | 0.01 - 10.0 | 速度过程噪声 |
| R_x, R_y, R_z | 0.01 - 10.0 | 测量噪声 |
| Mode | 0-2 | 预测模式：手动/自动/混合 |
| PredictTime | 0.0 - 5.0s | 手动预测时长 |

### Mask & Controls 窗口

HSV 颜色阈值和轮廓面积过滤。
